# 🧮 Mini Python Calculator

## 📖 Overview

The **Mini Python Calculator** is a beginner-friendly command-line application developed using Python. It allows users to perform basic arithmetic operations by entering two numbers and selecting an operator. This project demonstrates the use of Python fundamentals such as user input, conditional statements, arithmetic operators, and basic program flow.

---

## ✨ Features

* Perform Addition (+)
* Perform Subtraction (-)
* Perform Multiplication (*)
* Perform Division (/)
* Simple and easy-to-use command-line interface
* Displays an error message for invalid operators
